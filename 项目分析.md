## 项目分析报告

### 项目概览
**项目名称**: weber  
**技术栈**: TypeScript + OpenAI SDK + Ink (React for CLI)  
**类型**: AI CLI 编码助手（类似 Claude Code）

---

### 核心架构

#### 1. 双 API 模式
- **Responses API** (默认): `agentLoop()` 实现
- **Chat Completions API**: `agentLoopWithChatCompletions()` 实现
- 自动检测 `OPENAI_BASE_URL` 或手动设置 `OPENAI_API_MODE`

#### 2. 工具系统 (18+ 个工具)
| 类别 | 工具 |
|------|------|
| **Shell** | bash (带危险命令拦截、120s 超时) |
| **文件操作** | read_file, write_file, edit_file, glob, grep |
| **任务管理** | task_create/update/list/get/complete/fail/assign/block |
| **团队协作** | teammate_spawn/list/shutdown, message_send, lead_inbox |
| **技能系统** | load_skill |
| **MCP 集成** | list_mcp_resources, read_mcp_resource, mcp_call |

#### 3. UI 层
- 基于 Ink (React for CLI) 构建
- `UiBridge` 接口连接 agent 逻辑与 UI 更新
- 支持流式输出、思考过程显示、token 用量统计

#### 4. 高级特性
- **上下文压缩**: 防止上下文膨胀
- **团队模式**: 多智能体协作
- **技能系统**: 支持全局/本地技能覆盖
- **持久化**: 任务状态存储在 `.tasks` 目录

---

```
